# Gym Management Frontend

React frontend for the Gym Management System.

## Tech Stack

- React 18
- Vite
- TailwindCSS
- React Router
- Axios

## Setup

```bash
npm install
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env` and update:

```
VITE_API_URL=http://localhost:8000/api
```
